# Changelog — Aura Foods E-Commerce & ERP System

## v1.0.0 (2026-07-23)

### Initial Release
- Complete Next.js 16 application with TypeScript, Tailwind CSS, Prisma 7
- Public e-commerce website with product catalog, cart, and checkout
- Admin/ERP system with dashboard, product/order/inventory management
- SQLite database with seed data (8 products, 6 categories, sample orders)
- Custom session-based authentication
- Recharts-powered sales dashboard
- Responsive design for desktop and mobile

### What's Included
- 15+ public website pages and components
- 16+ admin pages for full ERP management
- 20+ API routes for data operations
- Reusable UI component library
- Comprehensive seed script with realistic data
- Full documentation suite (README, analysis, assumptions, known issues)

### Known Gaps
- No payment gateway
- No customer accounts
- No CSV import/export
- No batch/expiry tracking
- No supplier management
